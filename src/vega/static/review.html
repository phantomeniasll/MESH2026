⚠️ Sensor Fault Codes Explained

-99 °C / -1 % — Power-on sentinel. The sensor reports these values during boot / self-test before the first real measurement. The ESP32 may send a reading before the sensor is fully initialised. Harmless — subsequent readings should show real values.

-127 °C — I²C comms failure. The MCU couldn't reach the sensor over the bus. Check wiring (SDA/SCL) or sensor power.

0 % moisture in soil sensor — Sensor in air or unplugged. Capacitive sensors read ~0% when not inserted in soil.

These are hardware-level values, not bugs in VEGA. The API accepts them so you can see what's happening.
