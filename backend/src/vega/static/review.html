⚠️ Sensor Fault Codes Explained

-99 °C / -1 % — Power-on sentinel. Sensor reports these during boot/self-test before first real measurement.

-127 °C — I²C comms failure. MCU couldn't reach the sensor over the bus.

0 % moisture — Sensor in air or unplugged. Capacitive sensors read ~0% when not in soil.
