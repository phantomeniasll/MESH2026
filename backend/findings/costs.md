# Cost Research — Urban Tree Watering in Karlsruhe

**Research date:** June 26, 2026
**Sources:** DuckDuckGo searches, Karlsruhe city portal, German municipal data

---

## Karlsruhe Tree Inventory

- **~50,000 street trees** in Karlsruhe
- City spent **€2M on new trees** (Hildapromenade project, 2025/2026)

## Tree Costs

| Item | Cost |
|------|------|
| Plant new tree (small, with pit) | €300–600 |
| Plant new tree (medium/large) | €800–1,500 |
| Remove dead mature tree + replant + 3-5yr establishment watering | **€5,000–15,000** |
| Water truck operation | €80–150/hour (driver + fuel + water) |
| Per-tree watering (single session) | €5–15 |
| Per-tree watering (full summer, heat wave) | €30–90 |
| **City-wide watering (50k trees × €20/yr)** | **~€1M/year** |

## The Real Economics

Cities water on fixed schedules or citizen complaints — NOT data. A sensor network doesn't make watering *cheaper* per se. The actual wins:

1. **Dead tree prevention:** 100 extra trees kept alive = up to €1.5M avoided replacement cost
2. **Labor efficiency:** City crews only drive to trees that actually need water, not fixed routes
3. **Citizen labor:** Every tree a neighbor waters is one the city truck skips — free labor
4. **Tree protection cost:** Admitted — tree protection (guards, pruning, root management) costs more than watering. But those are fixed costs regardless of sensor data.

## Key Insight

The sensor doesn't save money on water. It saves trees. The pitch is: **"Trees that survive the next heat wave."**
